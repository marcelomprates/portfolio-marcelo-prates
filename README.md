## 📋 Sobre o projeto

Site de página única (single page) com navegação por âncoras, dividido em quatro seções interligadas por um menu fixo, visível em todas elas. O objetivo é apresentar, de forma clara e responsiva, minha formação, experiência e projetos.

## ✨ Funcionalidades

- **Quatro seções:** Sobre Mim, Formação, Portfólio e Contato.
- **Menu fixo** com navegação por âncoras e rolagem suave.
- **Tema claro/escuro** com persistência da preferência do usuário.
- **Menu responsivo** (hambúrguer) para dispositivos móveis.
- **Formulário de contato** com validação em JavaScript (campos obrigatórios e formato de e-mail) e simulação de envio com mensagem de confirmação.
- **Layout responsivo** (mobile first), adaptado a celular, tablet e desktop.

## 🛠️ Tecnologias

- **HTML5** — marcação semântica.
- **CSS3** — estilização com variáveis, Flexbox, Grid e Media Queries (sem frameworks).
- **JavaScript** — interatividade, validação e troca de tema (sem bibliotecas).

## ▶️ Como executar localmente

```bash

git clone https://github.com/marcelomprates/portfolio-marcelo-prates.git

# Acesse a pasta
cd portfolio-marcelo-prates
```

Depois, abra o arquivo `index.html` no navegador (duplo clique) ou use a extensão **Live Server** no Visual Studio Code.
